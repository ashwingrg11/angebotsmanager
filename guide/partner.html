# Partner

Partners are the entities which makes the Offers available. It is the company whose Offers are displayed. It is very simple with just formal information about their name, address and contacts within that company.

How to add a Partner?

To add a contact, simply click on the Partners button located on the navigation menu. This results in a dropdown with two options.

  • Add Partner

  • List/Edit Partners

Click on the Add Partner to load a page where fields used for the Partner are displayed. Enter the appropriate data and click Add button.

* Click to enlarge

# Fields

  • Select Project (Required) - A multiple select field to select Projects for the Partner. To select multiple Projects for a Partner, press Ctrl key on the keyboard and click on the desired projects

  • Partner Name (Required) - Enter the desired Partner Name.

  • Region (Optional) - A text field where Region for the Partner is entered.

  • Notes (Optional) - A text field to enter the notes (if any) for the partner.

# List/Edit Partners

This page will list all the Partners that have been added to the system. The contents of this page are in tabular form which contains columns whose brief description have been mentioned below



# Columns of the List Partner page

The Columns that appear in the List Partner page are

  • Partner Name - The name of the Partners appears in these columns.

  • Offers - Lists the Offers associated with the Partners. A Partner maybe associated with multiple Offers

  • Projects - Lists the Projects associated with the Partners. A Partner maybe associated with multiple Projects

  • Contacts - Lists the Contacts that have been assigned to the particular Partner. Multiple Contacts maybe assigned to a single Partner

  • Action - Contains three buttons whose functionalities have been discussed below.

A user will see three icons against each Partner name at the end of each row in this page.


    The edit button is used to edit the existing Partner details. Same validation rules are applied while editing Partners i.e. Select Project and Partner Name are required fields.

    The delete button is used to delete a Partner. However, do note that if a Partner is being used in other module, namely Offer , the Partner cannot be deleted. So, in order to delete a Partner that is been used in an Offer, a user first needs to delete the respective Offer and then proceed to delete the corresponding Partner.

    Clicking on this icon will enable the user to assign Contacts to the corresponding Partner. After clicking on this button, in the subsequent page that loads, a user needs to tick mark the appropriate contact which signifies that the said contact has been assigned to the Partner. Important factor to note here is that assigning contacts to the Partner is compulsory as not doing so will generate a validation error while adding Offer. Without assigning the contact to a Partner, a validation error will be generated while adding Offer asking a user to select the Primary Partner Contact. So before adding an Offer, a user needs to assign contact to the corresponding Partner. Also, note that the Primary Partner Contact so selected is the receiver of Activation Mail when an Offer is created. The Contacts assigned to Partner play a leading role in email communication.

# Partner Detail View

Clicking on the Partner name will give a detailed view of the particular Partner. It will provide a summary of the data(s) entered in the particular Partner.

# Access Rights

Access Rights for a creating/editing and deleting a Partner depends on the user and user type.This is discussed below.

  * Admin User

   An admin user can view/create/edit and delete (provided Partner has not been used in an Offer) all the Partners irrespective of the fact that he/she is assigned to the Project or not

  * General User

   A General User can only view/create/edit Partners for the Projects that he/she is assigned to. He/she cannot view the Partners for the Projects he has not been assigned to. A general user does not hold the right to delete a Partner

  * Viewer

   A Viewer can only see the Partner Name in the Offer description page. He/she does not hold the right to view all the details of the Partner or even create or modify the same
