# Offer Type

Offer Type fields are just different categories applicable to an Offer. They simply define a type for the Offers. However, they are not mandatory options when adding an Offer. There are basically three Offer Types in the system, namely

  • General Offer Type

  • Detailed Offer Type

  • Offer Type Category

Each Offer type can be categorically added into each of the above mentioned Offer Types.

# How to add an Offer Type into the system?

The Offer type can be added by going into the Additional Information and then clicking on the Offer Types . Three fields are visible when entering the Offer Type.

* Click to enlarge

# Fields

The field seen when adding an Offer Type is

  • Select Project (Required) - A dropdown list populated by the name of the Projects that have been entered into the system.

  • Select Offer Type (Required) - A dropdown featuring three categories of the above mentioned Offer Types

  • Offer Type Name (Required) - Name of the Offer Type to be added categorically into the above mentioned three Offer types

* Click to enlarge

List Offer Types

This page lists all the Offer types that have been added to the system. It is located just below the field for adding an Offer Type. A user will see two icons against each Offer Type name in this page. The Offer Types are categorized into columns of the respective Offer Type chosen.

# Columns of the List Offer Type page

The Columns that appear in the List Offer-Type page are

  • Project Name - The names of each Project used for entering the Offer Type

  • General - Lists the Offer Types entered into General Offer Type category

  • Detailed - Lists the Offer Types entered into Detailed Offer Type category

  • Category - Lists the Offer Types entered into Offer Type categories

 

  The edit button is used to edit the Offer type. Same validation rules are applied while editing Countries i.e. the Select Project, Select Offer Type and Offer Type Name is a required field. Each Offer Type can be edited. The category of Offer Type can also be changed between the above mentioned categories

    The delete button is used to delete an Offer Type. Any Offer Type can be deleted irrespective of the fact that they are being used in other modules.

* Click to enlarge

# Access Rights

Please note that an Offer Type can be created by an Admin User and a General User. A Viewer does not have access to these features. A General User has the ability to create and edit Offer type but he/she cannot delete it.
