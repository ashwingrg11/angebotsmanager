# Channels

Channels are based on the Projects. Channel can be added only if there are Projects. So, before adding a channel, a user needs to add Projects. Further channels are defined for a particular project

# How to add a channel?

The channels can be added by going into the Additional Information and then clicking on Channels. The page that opens contains a dropdown menu for Selecting Project and a field for adding Channel Name. Select the appropriate project and enter an appropriate channel name and click on Create Channel to add channel.

* Click to enlarge

# Fields

The fields seen when adding a channel are

  • Select Project - Dropdown (Required) - The projects created are populated in this dropdown

  • Channel Name (Required) - Give an appropriate name for the channel.

* Click to enlarge

# List Channel

In the same page of Create Channel page, there exists a list view of the channels that were or are to be created just below the fields for adding channels.

# Columns of List Channel

This page will list all the channels that have been added to the system. The Columns that appear in the List Project page are.

  • Name - Includes the name of the Channels

  • Project - Includes the name of the corresponding Project

  • Action - Gives a user the ability to Edit/Delete a channel

The icons seen at the end of each row in the List Channel page have been described below

   The edit button is used to edit the existing Channel details. Same validation rules are applied while editing Channels as for that of Adding Channels. Editing Channels enables a user to re-select the Project for the particular Channel or the name of the Channel.

   The delete button is used to delete a Channel. This module is dependent on other modules such as Placement. So as long as they are used in other modules, channels cannot be deleted. In order to remove a channel which is used elsewhere, a user needs to first remove the module using the Channel and then proceed to remove the Channel.

* Click to enlarge

# Access Rights

  * Admin User

An Admin user can add/edit/delete a channel

  * General User

A General User can create Channel only for the Project he is assigned to. He can also modify the details of the Channel. But he is not able to delete the Channel.
