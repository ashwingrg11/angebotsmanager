# Microsite Feedback Summary

Microsite2 Feedback Summary is the listing page of all the Microsite2 generated when an offer is created. This page is similar to that of Microsite Feedback Summary page except for the fact that it is only populated with Microsite2.

# Columns of the Microsite2 Feedback Summary page

  • Month - Lists the Month on which Microsite 2 was generated. Clicking on this month will open Microsite2 Detail Summary page.

  • Projects - Lists the Projects associated with the corresponding Offer and thus the Microsite2 so generated.

  • Partner Contacts - Lists the Partner contacts associated with the Offer whose Microsites were generated.

  • Offers - Lists the Offer for which the Microsite(s) were generated

  • Question1 Answer - Lists the answer of question 1 which was selected in Microsite 2 page. The answer, so selected, is in the form of a checkbox with a value of either 'Yes' or 'No'

  • Question 1 Comments - Lists the comments of question 1 so entered

  • Question 2 Comments - Lists the comment of question 2 so entered

  • Question 3 Ans - Lists the answer of question 3 so selected ('Yes' or 'No')

  • Question 3 Comments - Lists the comment of question 3 so entered

  • Extension Request - List whether the Extension Request was requested or not

  • Link To Microsite2 - Lists links as 2EN, 2DE. The preceding number signifies Microsite 2 and the succeeding characters signifies the language of the Microsite i.e. 2 is for Microsite 2 and EN is for English language. Clicking on these links will open the corresponding Microsite 2 page with the corresponding language.

# Access Rights

# Admin User

An Admin User can view all the details of the Microsite so generated.

# General User

A General User can only see the Microsite details of the Project or to be more specific, Project within an Offer, that he/she is assigned to.

# Viewer

A Viewer does not possess any right to view the Microsite data.
