# Call Center Briefing

Call center briefing is the summary of all offers for a specific month for a specific project. It includes all offers already valid (offer start date in prior month), offers with starting validity (offer start date within the month of the briefing document) or offers with a placement start date within the month of the briefing document). The offers shall be presented together with all pertinent description fields. The export is in the form of a word document.

# Conditions for export of the Briefing Document

  • If an offer validity period ends before the reporting start date, its detail will not be available in the word document.

  • If an offer is valid during the report start and end dates, its details are available in the word document.

  • If an offer starts after the report end date, but the placement starts prior to the report end date (as placement can start before an offer start date), its details are also available in the word document.

  • If an offer starts after the report end date and so does it's placement, then the details are not available in the word document.

# Here is an example to illustrate further

The user wants to generate a briefing that will be sent to the German call center at the end of January. The report shall cover the period February 5 (e.g. when the Q1 publication goes live) until March 31.
The generated report output would then include all the offers that fall with their validity within that date period.

  • Offer A (valid Dec 15 to Jan 31) will not be included.

  • Offer B (valid Dec 15 to Feb 15) will be included.

  • Offer C (valid April 1 to May 3 with a placement start date of March 20) will be included.

  • Offer D (valid April 1 to May 3) will not be included.

So conclusively, the word document will contain details of only those offers and also their placements that are valid during the reporting period.

# How to export a Briefing Document?

To export a briefing document, click on Reporting > Call Center Briefing which will open a page with teh following fields as discussed below -

# Fields

  • Select Project (Required) - Select the Project for which the briefing document is to be exported

  • Select Language (Required) - Select the language of the briefing document. The export is downloaded as per the language selected namely English and German

  • Date From (Required) - Select the starting date from which the briefing document is to be downloaded.

  • Date Until (Required) - Select the end date upto which the briefing document is to be downloaded

Clicking on the Export button will download a word file and all the details of the Offer are seen in the word document

# Access Rights

# Admin User

An Admin User can download briefing document for any project or offer.

# General User

A General User can only download the briefing document for the projects he/ she has been assigned to.

# Viewer

A Viewer does not possess any right to export the briefing document.
