# All Contacts Export

This reporting module basically allows JI Angebotmanager's users to download or export the report for contacts based on the Project he/she is associated with and also the placement. In this module, all the contacts associated with respective Projects and consequently their respective Offers populate the exported excel file. All the basic information related to the Contacts and also the Offers and Projects can be seen in that file.

# How to download All Contact Export file?

To download the All Contact document, click on Reporting > All Contacts Export. On this page, there are filter options on a page prior to generating the Excel export file. The filter options include

  • Projects - Projects shown in filter based on user eligibility and allowed for multiple selection. Also included is "N/A" for any unassigned partner contacts which is visible only to Admin users.

  • Placements - Placement is shown based on the selected Project(s) and is also allowed for multiple selection. Same as the project filter, also included is "N/A" for any unassigned partner contacts which is visible only to Admin users

# Access Rights

# Admin User

An Admin User can download All Contacts Export document for any project.

# General User

A General User can only download the All Contacts Export document for the projects he/ she has been assigned to.

# Viewer

A Viewer does not possess any right to export this file.
