# Activation Link

Activation Links page is the listing page for all the activation link that has been generated by the system. This page is mainly concerned with the activation links that are generated when an Offer is created. All the activation links so generated are listed on this page.

* Click to enlarge

# Columns of the Activation Links

The columns seen on this page have been briefly described below

  • Offer Title - The names of each Offers for which emails were created.

  • Partner Name - The partner lists that were used in the Offer.

  • Partner Contact Email - The email address of the Primary Partner Contact who received the activation link .

  • Activation Link - The link to activate an offer. Offer can be activated by two ways as mentioned below.

    • Clicking on the link as seen in the table of Activation Links page.

    • Clicking on the link received in email. This can be done by Primary Partner Contact.

  • Link Sent Out - Gives the status if the Activation link was sent out or not

  • Activation Status - Indicates whether the Offer is active or not.

# Access Rights

# Admin User

An Admin User can see all the Activation Links that were sent out.

# General User

A General User can see the Activation Links generated for only those Projects he/she is assigned to.

# Viewer

A Viewer cannot see any part of the activation links
