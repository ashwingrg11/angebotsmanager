# My Account

The "Angebotsmanager" (offer manager) allows a user to individually edit their account settings. Click on the located at the top right corner of the Angebotsmanager which will result in a dropdown. Click on the My Account option to edit individual user's info. Clicking on it will load a page with three tabs on top. These tabs have been briefly described below -

  • My Profile - Lists the detail summary of the user

  • Edit Profile - Allows the user to change their basic information such as Name, Last name, email etc. However, only an admin user can change the user type. Do note that if he/she does change the user type, then page reloads with the view of the corresponding user type selected. For instance, if an admin user decides to change his user type to that of a general user, then the page reloads and he can only use features that are accessible to a General User

  • Reset Password - Contains the functionality to reset the password of a user
