# Placement

A global list of Placement is maintained. These Placements form the basis for email communication and are used in Offer . An Offer can have multiple Placements. The Placement date determines when the Activation Email is to be sent to the Primary Partner Contact. To read more about when the email are sent, click on this link .

How to add a Placement?

To add a Placement, click on the Placement button located on the navigation menu. This results in a collapsible menu with two options.

  • Add Placement

  • List/Edit Placements

Click on the Add Placement to open a page where fields used for the placement are displayed. Enter the appropriate data as mentioned below and click Add button.

# Fields

  • Select Project (Required) - A dropdown field to select the Project for the Placement. This dropdown is populated as per the projects that exists in the system

  • Select Channel (Required) - A dropdown field to select the channel of the "Project" that is selected. Take note that the dropdown is populated only by the channels that have been associated with the particular Project, so selected

  • Placement Title (Required) - A text field where name for the Placement is entered.

  • Start Date (Required) - A text field to select the date from the Calendar that is displayed. This field determines when the Placement starts

  • End Date (Optional) - A text field to select the end date of the Placement from the Calendar that is dislayed. However, this field is optional and not specifying the same will not generate any validation error

# Columns of the List Placements page

The Columns that appear in the List Placement page are

  • Placements - The name of the Placements appears in these columns.

  • Projects - The Projects that have been used for the particular Placement are shown against each Placement name..

  • Channels - Lists the channels that that have been used for the Placement.

  • Start Date - The start date for a particular Placement are shown in this column.

  • Offers - Lists the Offers that have used the particular Placement. Please note that a placement may be used by multiple Offers so the list that populates may contain more than one Offer.

  • Actions - This column displays two icons against each Placement name for editing/deleting a Placement

Users can see two icons under the Action column whose brief functionality have been mentioned below

 

 The edit button is used to edit the existing Placement details. The validation rules are similar to that while Adding Placement . However, please make sure that if the End Date of the Placement needs to be specified, it needs to be a date which comes after the Placement Start Date.

   The delete button is used to delete a Placement. However, do note that if a Placement is being used in other modules such as Offer , the placement cannot be deleted. So, in order to delete a Placement, a user first needs to delete the respective module associated with the Placement and then proceed to delete the corresponding Placement.
