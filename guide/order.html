# Flowchart of JI Offer Tool

The general flow of Angebotsmanager is briefly described below

  • Create a contact

  • Create a client

  • Assign contacts to clients

  • Add Project categories as per desired

  • Create a Project

  • Create a Channel for the Project

  • Create a Partner and assign Contacts to it

  • Create Placement

  • Create templates & communication package for the emails to be generated (Optional)

  • Create Offer

  • Create Offer type for Offers (Optional)

  • View the various reporting data and the functionalities in the Reporting section

  • View the emails so generated in Partner Emails -> List/Edit Queue Emails section
